Parallax Effect Demo

This is a really simple implementation of the parallax effect. There are two "container" elements, the first one contains the image and the second one contains this text section.

The first container has a background image applied to it, with the CSS property background-attachment: fixed applied to it. As the name implies, this keeps the background image in place while the user scrolls.

The second container can be whatever you want, but typically it is used for the main content. There are no hard requirements for this container.
